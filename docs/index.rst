Lotus Documentation
=================================

Welcome to the documentation for Lotus, a cross-platform 3D game engine. Tutorials and guides for users and developers can
be found here, along with key API documentation. For a complete reference of the C++ code consult the
`Doxygen documentation <https://studiocentauri.github.io/lotus/doxygen/html/index.html>`_.

.. WARNING::
   Lotus is still in its early stages of development and is being actively worked on, and as a result the documentation
   might be outdated. If you find something that needs to be updated, please reach out to us.
